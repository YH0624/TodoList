# TodoListプロジェクト説明
  IDとメモ記入すると、メモ内容が登録される仕様
  登録されたメモ内容をタップすることで
	「更新」、「削除」、「キャンセル」も可能です。
	
# 開発環境
	Android Studio Ver3.5.2
	JAVA、SQLite(DBA)
